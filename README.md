## What is the PAKSAT HTS Terminal?

The **PAKSAT HTS User Terminal** is a compact satellite communication device that connects to the **PakSat-MM1 satellite** (launched May 2024), delivering high-speed internet to remote and underserved areas across Pakistan.

--------------------------

> **High Throughput Satellite (HTS)** = More bandwidth, lower cost per megabit.

## 🚀 Quick Start

```bash
# 1. Unpack the terminal
Terminal box includes:
- 0.98m satellite antenna (reflector)
- Feed horn + LNB
- BUC (Block Up Converter)
- Satellite modem
- Cables (coaxial, Ethernet)
- Power supply

# 2. Assemble the antenna
Mount reflector → Attach feed horn → Connect LNB/BUC

# 3. Point to PakSat-MM1
Azimuth:    Varies by city (see installation guide)
Elevation:  Varies by city
Polarization: Adjust for max signal

# 4. Power up modem
Connect to AC power → Wait for boot (2-3 minutes)

# 5. Access web UI
http://192.168.0.1
Username: admin
Password: admin
